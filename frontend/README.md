# AI Video Studio Pro (Frontend)

This is the frontend dashboard for the AI Video Studio Pro project. It allows users to upload media and interact with AI-generated transcripts, summaries, and content tools.

## Prerequisites
- Node.js (v18 or higher recommended)
- The backend server must be running (see backend/README.md)

1. Open a terminal in this `frontend` directory.
